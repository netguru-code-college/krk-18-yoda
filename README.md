## About

YodaDigitize is an app that lets you store your personal documents online, and manage them in categories.
Unprint what you printed. Join the cloud side!

* Ruby version
2.5.1

* Rails version
5.2.0

* System dependencies
Redis

## Setup

* Configuration
`bundle install`

* Database creation
`rails db:setup`

* Run background jobs
`bundle exec sidekiq`
